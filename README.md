# re-natal-esp32control-app

A Clojure library designed to control car over BLE connection.

# Usage

For a terminal on linux.
```
npm install
sudo npm install rnpm -g
rnpm link
react-native start
```

Another terminal on linux.
```
react-native run-android
```
